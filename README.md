# NEET PG 2025 — Counselling Analysis

## KEA Counselling Analysis

Course-wise analysis of KEA (Karnataka Examinations Authority) NEET PG 2025 counselling allotment data across three rounds.

Each report includes:
- **GM & GMP Min/Max Rank Summary** — one row per course with absolute min and max All India Rank
- **College-wise Rank Tables** — min/max All India Rank per college, for GM and GMP categories
- **College-wise Fee Tables** — min/max course fees per college, for GM and GMP categories
